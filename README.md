# Proposta Comercial — Studio X

Site de apresentação comercial da **Studio X**: conteúdo visual, vídeo, campanhas e estratégia para marcas venderem mais.

Página única, responsiva, com estética glassmorphism cinematográfica.

## Conteúdo

- **index.html** — estrutura da proposta (Hero, Serviços, Como funciona, Plano, Contato)
- **styles.css** — design (fundo cinematográfico, cards de vidro, tipografia serif + mono)
- **script.js** — link do WhatsApp, animações de scroll
- **img/** — imagens de fundo (geradas por IA)
- **Proposta-StudioX.pdf** — versão em PDF pronta para envio

## Rodar localmente

```bash
python3 -m http.server 4601
```

Abrir http://localhost:4601
